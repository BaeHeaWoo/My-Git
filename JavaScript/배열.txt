배열의 추가
	var li = ['a','b'];
	li.push('c') -> push는 li라는 배열에 소속됨 후에 c를 꼬리에 추가  
            lli.concat(['f', 'g']); -> 여러가지 인지 추가할때 사용 인자값은 배열 그후 결과를 반환 한다 하지만 실제로 반영이 안됨 
            li.unshift('h') -> unshift는 li라는 배열에 소속됨 후에 h를 머리에 추가
            li.splice -> 배열의 특정구가을 추출하거나, 특정구간에 특정 배열을 추가함
           array.splice(index, howmany, element....) 
           -> index.필수(number)
           -> howmany 필수(number) index에서부터 제거될 원소들을 수 (0이면 삭제 안됨)
           -> element옵션 index howmany 사이에 추가될 값
           return index와 howmany 범위의 값 return 

배열의 제거
         var li = ['a', 'b', 'c', 'd', 'e'];
         li.shift() -> shift는 li라는 배열에 소속됨 후에 를 머리를 삭제
         li.pop() -> pop은 li라는 배열에 소속됨 후에 꼬리를 삭제
